# Quran Navigator
this app helps you to get every verse in Quran on multiple languages (21 languages)
-Original_Arabic
-Chinese
-French
-engyusufali
-Persian
-Japanese
-Norwegian
-Bangla
-Malayalam
-Dutch
-Urdu
-Turkish
-Italian-Piccardo
-Russian
-Portuguese
-Swedish
-German
-Tamil
-Uzbek
-Indonesian
-Korean
  
and you can search by word on specifying the language



## Installation
Make sure that you have these requirement installed:
    -NodeJs
    -Angular

Step 1. Run the API
on the root folder  enter:
```
npm install
node server
```
Step 3. Run the app
on the quran-navigator folder  enter:
```
npm install
ng serve
```
